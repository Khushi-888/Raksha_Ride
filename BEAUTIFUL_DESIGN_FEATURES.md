## 🎨 Design Elements

### Color Palette
```
Primary: #FFC107 (Golden Yellow)
Primary Dark: #FF9800 (Orange)
Secondary: #667eea (Purple)
Dark: #0f172a (Navy Blue)
Dark Light: #1e293b (Slate)
```

### Typography
- **Font**: Inter (Google Fonts)
- **Weights**: 300, 400, 500, 600, 700, 800
- **Sizes**: Responsive scale from 12px to 48px

### Spacing System
- **Base unit**: 8px
- **Scale**: 8px, 16px, 24px, 32px, 40px, 48px
- **Consistent gaps** throughout

### Border Radius
- **Small**: 8px
- **Medium**: 12px
- **Large**: 16px
- **Extra Large**: 24px

## 🎯 User Flow

### Welcome Screen
1. User lands on beautiful hero page
2. Sees feature cards
3. Clicks "Sign In" or "Join Now"

### Login
1. Form slides in smoothly
2. User selects role (Passenger/Driver)
3. Enters credentials
4. Gets visual feedback
5. Redirects to dashboard

### Registration
1. Form appears with animation
2. User selects role
3. Fills in details
4. Clicks "Send OTP"
5. OTP section reveals
6. Enters OTP
7. Account created
8. Redirects to login

## 🎨 Customization

Want to change colors? Edit `static/css/main.css`:

```css
:root {
    --primary: #FFC107;  /* Change this */
    --primary-dark: #FF9800;  /* And this */
    /* ... */
}
```

All colors update automatically!
